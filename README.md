# 设计模式教程

面向初学者和进阶开发者的设计模式教程，每个模式用 TypeScript 和 Python 双语示例讲解。

## 目录结构

```text
fundamentals/      前置基础（2 篇）
creational/       创建型模式（6 篇）
structural/        结构型模式（7 篇）
behavioral/        行为型模式（11 篇）
architecture/      架构学习路线
ai-engineering/    AI 工程能力
learning-progress.md  学习进度追踪
AGENTS.md          写作规范和教学原则
```

## 推荐学习顺序

1. 前置基础：面向对象基础 → 设计原则
2. 创建型：单例 → 简单工厂 → 工厂方法 → 抽象工厂 → 建造者 → 原型
3. 结构型：适配器 → 装饰器 → 代理 → 外观 → 桥接 → 组合 → 享元
4. 行为型：策略 → 模板方法 → 观察者 → 责任链 → 命令 → 状态 → 迭代器 → 中介者 → 备忘录 → 访问者 → 解释器
5. 架构学习：架构学习路线 → AI 时代架构方法论 → 分层架构 → 依赖倒置落地 → Clean Architecture → 六边形架构 → DDD 入门 → 前端架构 → MVC/MVP/MVVM/Flux/VIP → 后端架构 → 模块化单体 → CQRS → 事件驱动 → BFF → 微服务边界 → ADR → 测试策略 → 架构评审 → 演进式架构 → 综合实践
6. AI 工程能力：代码坏味道与重构 → 模式选择 → AI 代码评审 → Prompt 到架构方案 → 上下文工程 → 测试驱动 AI 重构

## 笔记特点

- 先讲问题，再讲模式
- 每篇包含生活类比、适用/不适用场景、优缺点
- TypeScript 和 Python 双语示例，对比语言差异
- 每篇附练习题和今日总结
